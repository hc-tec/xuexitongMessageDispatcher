## 仿 Django 设计思想简易后端框架 + 区块链技术(PoW认证)

### 目录结构
```
├─Blockchain         区块链类设计
│  └─Transaction     交易类设计
├─config             配置文件目录
├─Handlers           处理业务逻辑的类
├─Main               主程序入口
├─Store              储存类
├─TcpServer          简易服务器
│  └─Response        响应类
└─utils              工具类目录
    ├─Email          邮箱发送
    ├─Exception      异常
    ├─Request        仿 Python 的 requests 封装的请求类
    └─Serializer     序列化
```
